사용법

이 도구 사용법

  1. 접사 검색하기 — 검색창에 원하는 접사를 한국어로 입력합니다 (예: “치명타”)
  2. 접사 클릭 — 검색 결과에서 원하는 접사를 클릭하면 출력 바에 자동으로 추가됩니다
  3. 조합하기 — 여러 접사를 조합하려면:
    첫 번째 접사 클릭 → 또는 또는 그리고 버튼 클릭 → 두 번째 접사 클릭
  4. 프리셋 사용 — 검색창 옆 프리셋 버튼을 눌러 LP, T7 등을 빠르게 추가
  5. 복사 & 붙여넣기 — 복사 버튼을 눌러 결과를 복사하고, 게임 내 창고 검색창에 붙여넣기

기본 검색

창고 검색창에 텍스트를 입력하면 아이템 툴팁에서 해당 텍스트를 찾습니다.

예: 화염 저항 → 화염 저항이 있는 모든 아이템

정규표현식 (Regex)

/ 로 감싸면 정규표현식으로 검색합니다 (대소문자 무시).

예: /치명타/ → “치명타”가 포함된 모든 아이템

예: /화염|냉기|번개/ → 화염, 냉기, 또는 번개가 포함된 아이템

프리셋

  • LP3+ — 잠재력 3 이상인 유니크 아이템
  • T7 — T7 접사가 하나라도 있는 아이템
  • 2T7 — T7 접사가 2개 이상인 아이템 (더블 T7)
  • WW20+ — 직공의 의지가 20 이상인 아이템
  • CoF — 운명의 고리 태그가 있는 아이템
  • MG — 상인 길드 태그가 있는 아이템

연산자

  • & (그리고) — 양쪽 조건 모두 만족: T7&/치명타/
  • | (또는) — 어느 한쪽만 만족: LP3+|WW20+
  • + (이상) — 지정한 수 이상: T6+ = T6 이상
  • - (이하) — 지정한 수 이하: WW12- = WW 12 이하

실전 예제

클릭하면 출력 바에 자동으로 입력됩니다

/치명타 확률/
치명타 확률 접사가 있는 아이템
T7&/체력/
T7 접사 + 체력이 있는 아이템
LP3+|WW20+
잠재력 3+ 유니크 또는 직공의 의지 20+
T7&/화염 저항/
T7 접사 + 화염 저항이 있는 아이템
2T7
T7 접사가 2개 이상인 숭고 아이템
/화염|냉기|번개/&/저항/
원소 저항이 있는 아이템
T6+&/방어력/
T6 이상 접사 + 방어력이 있는 아이템
CoF&T7
운명의 고리 + T7 접사

참고사항

  • 프리셋(T7, LP 등)은 영어로 입력해야 합니다 — 게임 엔진이 영어 키워드를 인식합니다
  • 접사 검색은 한국어로 합니다 — 한국어 패치 적용 시 툴팁이 한국어이기 때문입니다
  • 티어 프리셋(T5+ 등)은 특정 접사가 아닌 아이템의 아무 접사에 적용됩니다
  • 게임 검색창에는 최대 30자까지 입력할 수 있습니다 — 이 도구는 자동으로 접사 이름을 축약하여 글자 수를 절약합니다
